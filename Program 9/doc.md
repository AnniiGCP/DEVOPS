# BCSL657D – DevOps Lab

## Program 9: Introduction to Azure DevOps (Setup)

---

## 🎯 Aim

To create an Azure DevOps account, organization, and project.

---

## 📋 Prerequisites

* Microsoft account
* Internet connection
* Web browser

---

# ⚙️ Step 1: Open Azure Portal

* Open browser
* Go to:

```text
https://portal.azure.com
```

* Sign in with Microsoft account

---

# ⚙️ Step 2: Open Microsoft Cloud Menu

* Click **top-left menu (Microsoft Cloud menu)**
* Select:

```text
DevOps
```

👉 This will redirect to Azure DevOps portal

---

# ⚙️ Step 3: Create Organization

* Click **New Organization**

Enter:

```text
Organization Name: ksit-devops
Region: India
```

* Click **Continue**

---

# ⚙️ Step 4: Create Project

Enter details:

```text
Project Name: student-app-devops
Visibility: Private
Version Control: Git
Work Item Process: Basic
```

* Click **Create Project**

---

# ⚙️ Step 5: Explore Services

* Boards → Work items
* Repos → Source code
* Pipelines → CI/CD
* Test Plans → Testing
* Artifacts → Packages

---

# ⚙️ Step 6: Initialize Repository

* Go to: Repos → Files
* Click **Initialize**

👉 This creates:

* README.md
* main branch
* First commit

---

# ⚙️ Step 7: Copy Clone URL

* Go to Repos
* Copy HTTPS URL for future use

---

# ✅ Output

* Azure DevOps account accessed
* Organization created
* Project created
* Repository initialized

---

# 🧠 Conclusion

Azure DevOps environment is successfully set up using the Microsoft Azure portal navigation, and is ready for further DevOps operations.
